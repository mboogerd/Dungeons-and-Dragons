# 🌑 Core Principles
1.	Resonant, breathy sounds — they don’t have mouths in the human sense, but vibrating gills, spores, and threads:
	•	Nasals: m, n, ng (think of “hum,” “drone”)
	•	Liquids: l, r (flowing, connective)
	•	Voiced stops: b, d, g (soft, thudding, subterranean)
	•	Almost no sharp hissing consonants (s, z, ch) — too airy/“surface.”
2.	Muffled vowels, deep resonance — earthy, humid, low-pitched:
	•	u, o, a as primary vowels (rounded, cavernous)
	•	i, e appear as glimmers, like spore-lights or sharper shifts.
3.	Morphemes tied to life cycles
	•	mu / my / ma → “we / life / breath”
	•	lo / lu → “root / hold / ground”
	•	nar / nur → “cycle / return”
	•	ghum / gom → “network / chorus”
	•	thil / thol → “thread / strand”
4.	Structure
	•	Words tend to be short, 1–2 syllables (they’re more vibrations than words).
	•	Meaning is contextual and layered: “Mu” might mean “we” in one setting, “life” in another, “the cycle” in a ritual. Outsiders struggle because one word is all of those things at once.

# 🍄 Fungal Root Lexicon
Core Concepts
	•	Mu → we, life, breath, being (the most fundamental word)
	•	Nar → cycle, return, rebirth
	•	Lo → root, grounding, anchor
	•	Thil → thread, strand, link
	•	Ghum → chorus, hum, network, resonance
Growth & Change
	•	Sul → sprout, beginning, growth
	•	Mor → decay, rot, transformation
	•	Vun → fruiting, emergence, showing self
	•	Lum → small life, spark, tender shoot
Mystic & Spirit
	•	Orr → shadow, veil, ghost
	•	Thol → deep, beneath, hidden
	•	Ruun → memory, echo, trace
	•	Ghal → spirit, essence, animating force
Communal & Individual
	•	Wem → all, whole, togetherness
	•	Ta → self, spark of individuality
	•	Drul → apart, severed, cut-off

# 🌱 Word composition
🍂 Example Combinations
	•	Munarlo → “We are the cycle rooted” (could mean “eternal life,” “the network,” or “our people”)
	•	Thilmu → “life-thread,” also “bond,” also “me” when asserting individuality
	•	Ghumlo → “rooted network,” “the chorus below”
	•	Narghum → “cycle of voices,” “renewal”
	•	Lumun → “small life,” often used as a tender word (a sapling, or a youngling)
	•	Ghumloruun → “network-root-memory” → ancestral voices
	•	Sulvun → “sprout-fruit” → coming-of-age, blooming into being
	•	Orrmor → “ghost-rot” → death, dissolution, haunting
	•	Tawe → “self-all” → paradoxical phrase for individuality within the whole

# 🍄 20-Word Fungal Core
Being & Identity
	1.	We / Life → Mu (breath, being, essence)
	2.	Network / Chorus → Ghum (hum, resonance, binding)
	3.	Self / Spark → Ta (rare, dangerous individuality)
	4.	Apart / Cut off → Drul (severed, isolated, dead to the web)
Growth & Cycle
	5.	Cycle / Return → Nar (rebirth, renewal, endless turn)
	6.	Root / Anchor → Lo (ground, stability, connection)
	7.	Sprout / New beginning → Sul (youth, first life)
	8.	Fruit / Bloom / Emergence → Vun (to show, to ripen, to reveal)
Change & Decay
	9.	Rot / Transformation → Mor (death that feeds, creative decay)
	10.	Shadow / Ghost → Orr (veil, half-being, spectral echo)
	11.	Memory / Trace → Ruun (what lingers, what remains in the network)
	12.	Deep / Hidden / Below → Thol (subterranean, secret, unknowable)
Natural World
	13.	Tree / Plant ally → Bral (the green partners, hosts)
	14.	Stone / Earth → Kar (bedrock, cavern, foundation)
	15.	Water / Flow → Nelu (stream, pulse, moist breath)
	16.	Fire / Destruction → Zha (heat, burning, cleansing, feared)
Relational & Emotional
	17.	Together / Whole → Wem (all-as-one, harmony)
	18.	Love / Care / Nourish → Thun (feeding, tending, cherishing)
	19.	Fear / Dread → Gro (a deep groan, danger, rupture)
	20.	Hope / Light / Spark → Lir (faint glow, bioluminescence, promise)
